## 五、技术栈总览

### 5.1 技术栈矩阵

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端框架** |
| Web前端 | Vue 3 + Element Plus | 主流响应式框架 |
| 移动端 | Uni-app / React Native | 跨平台小程序/App |
| 地图组件 | AMap /腾讯地图 | 考勤打卡定位 |
| **后端框架** |
| 核心框架 | Spring Boot 3.x + Spring Cloud | Java 17+ |
| API网关 | Spring Cloud Gateway | 统一入口 |
| 服务注册 | Nacos | 注册中心+配置中心 |
| 限流熔断 | Sentinel | 高可用保障 |
| 分布式事务 | Seata | TCC/Saga模式 |
| **消息队列** |
| 高吞吐量 | Apache Kafka | 审批事件/聊天消息/报表任务 |
| 可靠消息 | RabbitMQ | 邮件发送/通知/延迟任务 |
| **数据存储** |
| 关系型 | MySQL 8.0 | 主备库+读写分离 |
| 缓存 | Redis Cluster | 多级缓存 |
| 对象存储 | MinIO | S3兼容，私有化部署 |
| 向量数据库 | Milvus | RAG知识库 |
| 搜索引擎 | Elasticsearch | 日志/全文检索 |
| **AI能力** |
| 生文模型 | GPT-4o / Claude-3.5 / Kimi / 文心 / 智谱 | 多模型配置 |
| 生图模型 | DALL-E 3 / Midjourney / Stable Diffusion | 按功能分配 |
| 生视频模型 | Sora / Runway Gen-3 / 智谱Video | 按功能分配 |
| 语音模型 | ElevenLabs / Azure TTS / 阿里语音 | 按功能分配 |
| OCR | PaddleOCR + 阿里云OCR | 发票/行程单识别 |
| **流程引擎** |
| 工作流 | n8n | 可视化流程编排 |
| **基础设施** |
| 容器编排 | Kubernetes | 弹性伸缩 |
| 容器镜像 | Docker + Harbor | 镜像管理 |
| CI/CD | GitLab CI + ArgoCD | GitOps流水线 |
| 监控 | Prometheus + Grafana | 可观测性 |
| 日志 | Loki + Kibana | 日志聚合 |
| 链路追踪 | Jaeger | 分布式追踪 |
| **集成对接** |
| 打车平台 | 滴滴企业版 / 高德 / 首汽 / 曹操 | API对接 |
| 企业邮箱 | 钉钉邮箱 / 企微邮箱 / 网易 / 腾讯 / 263 | IMAP/SMTP |
| 办公IM | 钉钉 / 企业微信 / 飞书 | 消息推送 |
| 消息推送 | 极光 / 小米 / APNs | 离线推送 |
