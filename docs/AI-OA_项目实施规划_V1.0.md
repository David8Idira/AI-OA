## 一、项目概览

### 1.1 项目背景

基于RuoYi开源框架，整合OCR识别、n8n流程引擎、AI大模型能力，打造新一代智能化OA系统。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 财务审批自动化 | OCR发票+打车行程单自动识别 |
| 智能报表生成 | 周刊/月刊/年刊AI自动生成 |
| AI对话助手 | 多模型AI侧边栏 |
| 企业即时通讯 | WebSocket实时聊天 |

## 二、技术架构

### 2.1 后端架构（9个微服务）

```
aioa-gateway (8080)  ──┐
                      ├── 路由 + 鉴权
aioa-system (8081)  ──┼── 用户/部门/角色
aioa-workflow (8082) ─┼── 审批流引擎
aioa-ai (8083)      ──┼── AI对话+配额
aioa-im (8084)      ──┼── 即时通讯
aioa-ocr (8085)     ──┼── OCR识别
aioa-reimburse (8086) ─┼── 费用报销
aioa-report (8087) ──┼── 智能报表
aioa-common         ──┘── 公共基础
```

### 2.2 前端架构

| 组件 | 技术 |
|------|------|
| 管理后台 | Vue3 + TypeScript |
| 组件库 | Element Plus |
| HTTP | Axios |
| 状态管理 | Pinia |
| 路由 | Vue Router |

### 2.3 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Spring Boot 3.x |
| ORM | MyBatis Plus |
| 网关 | Spring Cloud Gateway |
| AI | OpenAI / Claude / Moonshot |
| 数据库 | MySQL / Redis |
| 消息队列 | Kafka / RabbitMQ |
