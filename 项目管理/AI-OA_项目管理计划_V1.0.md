## 一、项目概述

### 1.1 项目目标

基于RuoYi开源框架，开发一套智能化OA系统，实现：
- 财务审批自动化（OCR识别 + n8n流程引擎）
- 智能报表生成（周刊/月刊/年刊自动生成 + AI生图/视频）
- AI对话助手（侧边栏实时问答 + 多模型配置）
- 流程管理中心（n8n可视化编辑集成）
- 低代码可视化审批流（零代码配置 + 流程设计器 + 规则引擎）

### 1.2 项目范围

| 包含 | 不包含 |
|------|--------|
| RuoYi基础框架 | RuoYi-Vue3前端重写 |
| 财务审批模块 | 第三方支付集成 |
| OCR识别集成 | 硬件设备对接 |
| n8n流程引擎 | n8n源码修改 |
| AI对话助手 | 硬件/IoT集成 |
| 智能报表生成 | 线下活动管理 |
| 低代码可视化审批流 | 客户定制功能 |
| 测试与部署 | 第三方系统集成 |

### 1.3 项目约束

| 约束类型 | 说明 |
|----------|------|
| 时间约束 | 总工期16-20周 |
| 预算约束 | 开源优先，AI API按量付费 |
| 技术约束 | 基于RuoYi生态，Java+Vue3 |
| 人力约束 | 核心开发2-3人 |
