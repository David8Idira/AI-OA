# LRuoYi-OA

> 基于 RuoYi + OCR + n8n + AI 的智能化OA系统

## 项目简介

LRuoYi-OA 是一款基于 RuoYi 开源框架开发的智能化OA系统，整合了OCR识别、n8n流程引擎、AI大模型能力，实现财务审批自动化和智能报表生成。

## 核心功能

### 🔥 财务审批自动化
- OCR发票识别（PaddleOCR + 阿里云OCR）
- **打车行程单对接**（滴滴企业版/高德/首汽约车/曹操出行）
- n8n可视化审批流程
- 自动审批规则引擎
- 多级审批 + 实时知会

### 📊 智能报表生成
- 周刊/月刊/季刊/半年刊/年刊自动生成
- AI生图（封面/插图）
- 多渠道分发（邮件/飞书/钉钉/企业微信）

### 🤖 AI对话助手
- 自动隐藏侧边栏对话框
- 多模型配置（OpenAI/Claude/国产模型）
- RAG知识库问答
- 会话历史管理

### ⚙️ 流程管理中心
- n8n可视化编辑器集成
- 流程模板市场
- 执行监控与告警

### 📬 企业邮箱集成 ⭐新增
- **钉钉邮箱/企业微信邮箱/网易企业邮/腾讯企业邮**
- 统一配置界面
- 邮件代收与业务关联

### 🛠️ 增强功能 ⭐新增
- **低代码表单构建器**（参考泛微建模平台）
- **移动端适配**（响应式 + 小程序）
- **知识管理**（文档库/协同编辑/全文搜索）
- **目标管理**（OKR/KPI）
- **考勤管理**（GPS/WiFi打卡）
- **合同管理**（模板/审批/用章）

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Java 17+ / Spring Boot 3.x |
| 前端 | Vue 3 + Element Plus |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis |
| OCR | PaddleOCR + 阿里云OCR |
| 流程引擎 | n8n |
| AI能力 | OpenAI / Claude / 国产模型 |
