# LRuoYi-OA

> 基于 RuoYi + OCR + n8n + AI 的智能化OA系统

## 项目简介

LRuoYi-OA 是一款基于 RuoYi 开源框架开发的智能化OA系统，整合了OCR识别、n8n流程引擎、AI大模型能力，实现财务审批自动化和智能报表生成。

## 核心功能

| 模块 | 功能 | 技术方案 |
|------|------|----------|
| 🔥 财务审批 | OCR发票 + 行程单对接 + n8n审批流 | PaddleOCR + 阿里云OCR |
| 📊 智能报表 | 周刊/月刊/年刊 + AI生图/视频 | GPT-4o + DALL-E 3 |
| 🤖 AI助手 | 侧边栏对话 + 多模型配置 | 多模型按功能分配 |
| 💬 企业聊天 | 文字/图片/语音/截图 + 来信提醒 | WebSocket + Kafka |
| 📬 企业邮箱 | 钉钉/企微/网易/腾讯邮箱集成 | IMAP/SMTP |
| ⚙️ 流程中心 | n8n可视化编辑器 + 模板市场 | n8n嵌入式 |
| 🛠️ 增强功能 | 低代码/移动端/知识库/考勤/合同 | 竞品优秀功能整合 |

## 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端 | Java 17+ / Spring Boot 3.x / Spring Cloud |
| 前端 | Vue 3 + Element Plus |
| 数据库 | MySQL 8.0 (主备库+读写分离) |
| 缓存 | Redis Cluster (L1+L2多级) |
| 消息队列 | Kafka (高吞吐) + RabbitMQ (可靠消息) |
| 文件存储 | MinIO (S3兼容，私有化部署) |
| AI能力 | 生文(GPT-4o/Claude/Kimi) + 生图(DALL-E/MJ/SD) + 生视频(Sora/Runway) |
| OCR | PaddleOCR + 阿里云OCR |
| 流程引擎 | n8n |
| 容器化 | Docker + Kubernetes |
| CI/CD | GitLab CI + ArgoCD |
