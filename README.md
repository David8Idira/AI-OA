# LRuoYi-OA

> 基于 RuoYi + OCR + n8n + AI 的智能化OA系统

## 项目简介

LRuoYi-OA 是一款基于 RuoYi 开源框架开发的智能化OA系统，整合了OCR识别、n8n流程引擎、AI大模型能力，实现财务审批自动化和智能报表生成。

## 核心功能

### 🔥 财务审批自动化
- OCR发票识别（PaddleOCR + 阿里云OCR）
- **打车行程单对接**（滴滴企业版/高德/首汽约车/曹操出行）
- n8n可视化审批流程
- 自动审批规则引擎
- 多级审批 + 实时知会

### 📊 智能报表生成
- 周刊/月刊/季刊/半年刊/年刊自动生成
- AI生图（封面/插图）
- 多渠道分发（邮件/飞书/钉钉/企业微信）

### 🤖 AI对话助手
- 自动隐藏侧边栏对话框
- 多模型配置（OpenAI/Claude/国产模型）
- RAG知识库问答
- 会话历史管理

### 🔧 AI大模型配置 ⭐新增
- **多能力支持**：生文/生图/生视频/语音
- **功能模块分配**：按业务功能配置不同模型
- **模型路由**：主备模型 + 故障自动切换
- **费用统计**：各模型使用量/费用看板

### ⚙️ 流程管理中心
- n8n可视化编辑器集成
- 流程模板市场
- 执行监控与告警

### 💬 企业内部聊天 ⭐新增
- 文字/图片/语音/截图/文档消息
- 单聊/群聊/部门群
- 来信提醒（桌面通知/声音/推送）
- 已读未读状态同步

### 📬 企业邮箱集成 ⭐新增
- **钉钉邮箱/企业微信邮箱/网易企业邮/腾讯企业邮**
- 统一配置界面
- 邮件代收与业务关联

### 🛠️ 增强功能 ⭐新增
- **低代码表单构建器**（参考泛微建模平台）
- **移动端适配**（响应式 + 小程序）
- **知识管理**（文档库/协同编辑/全文搜索）
- **目标管理**（OKR/KPI）
- **考勤管理**（GPS/WiFi打卡）
- **合同管理**（模板/审批/用章）

### 部署方式

```bash
# Docker Compose 本地开发
docker-compose -f deploy/docker/docker-compose.yml up -d

# Kubernetes 生产部署
kubectl apply -f deploy/k8s/

# Helm 一键部署
helm install lruoyi-oa ./deploy/helm -n oa-system -f values-prod.yaml
```

### 核心配置

- **HPA**: 2-20副本，CPU>70%触发扩容
- **健康检查**: liveness/readiness/startupProbe
- **反亲和**: Pod分散到不同节点
- **资源限制**: requests/limits分离

### 部署配置

| 服务类型 | 最小副本 | 最大副本 | 扩容阈值 |
|----------|----------|----------|----------|
| API Gateway | 3 | 20 | CPU 70% |
| 系统服务 | 3 | 15 | CPU 70%, Memory 80% |
| AI 服务 | 2 | 10 | CPU 70% |
| OCR 服务 | 2 | 10 | Queue > 10 |
